Machine Learning for Trading

这么课程在Coursera上面也有。整个课程分为两个部分，但是在Cousera上面只放了 前半部分 。我整理了这部分的 作业 答案

这个是他们的 教学主页 ，使用QSTK(Quant Software Toolkit)来做教学，在 github 上面也有源代码。

这么课程实践性非常强，即便是学完了如果没有实战的话，理解就只能停留在表面。

个人感觉，在实践的时候需要很多金融/交易方面的知识才行，否则无法下手。这个东西非常需要领域知识，只会靠机器学习和统计学习的人是不会成功的。
