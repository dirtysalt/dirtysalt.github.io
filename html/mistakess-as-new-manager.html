Some mistakes I made as a new manager

Some mistakes I made as a new manager | benkuhn.net

感觉还是中文好读，虽然里面会出现某些词不达意的翻译，通过看原文还是可以纠正过来的。

总结下来是7条，每条我都加点自己的注释。

  • 没有即时反馈。这个必须是心态上最好调整，然后通过其他方式来获得间接反馈。
  • 坚持关键路径。这个就是没事不要去写代码，但是如果下面人写不过来的时候，还需要自己上。
  • 错误的管理范围。如果下面是不太熟悉的新手，那么最好还是微管理；但是如果已经是老手了，那么就直接要结果。不要搞一刀切。
  • 没有办法快速决断。这个估计还是需要找人一起讨论，寻找帮助。
  • 依然按照优先级做事。manager要做的事情更多了，但是按照优先级做事这点准则依然成立。
  • 不要怕直接询问。这个估计还是碍于面子吧，要把事情做好，也不要让自己受委屈，还是要直接说出来。

总之我觉得写得挺诚恳的。
