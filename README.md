## 目录安排
- `src` **org-mode** 组织的源文件
- `html` 从源文件生成的html文件
- `images` 引用的图片文件
- `scripts`
   - `publish/publish.el` 发布脚本
   - `pp-html` 后期处理html文件，包括添加disqus评论以及GA等代码
   - `check-ref-images.py` 检查org文件对images目录下面引用情况
- `sitemap.xml/.txt` 站点导图
- `404.html/index.html`
- `themes` 站点样式配置
