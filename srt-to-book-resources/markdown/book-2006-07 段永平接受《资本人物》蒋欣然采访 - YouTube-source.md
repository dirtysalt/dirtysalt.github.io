# Chapter 1: 股神午餐、欲速不达与网易传奇

2006年6月30日，美国西岸时间上午11时，一场紧张的竞拍在eBay上进行着。这场竞拍的奖品，是与股神沃伦·巴菲特共进午餐的宝贵机会。经过数轮激烈的出价，一个网名叫“Fastest Slow”的竞拍者，最终以62.01万美元的天价，赢得了这次午餐。次日，人们赫然发现，这位神秘的“Fastest Slow”正是步步高公司的创始人——段永平。

当被问及为何取名“Fastest Slow”时，段永平微笑着解释，这其实是“欲速不达”的意思，也是他做人做事的根本哲学：不求快，但求安全。他坦言，这次身份的曝光纯属意外，因为他在eBay上注册时，习惯性地使用了自己常用的ID，并未多想。

段永平参与竞拍，原因有二。首先，他对巴菲特的投资理念怀有深深的敬意。作为一名资深的企业家，他发现巴菲特的投资理论与他多年经营企业的经验不谋而合，让他茅塞顿开。他认为，如果没有接触巴菲特，他可能根本不敢涉足投资领域。因此，他渴望能与巴菲特面对面交流，解决一些投资中的细枝末节，甚至替一位爱借钱的朋友问问“手里有钱没目标，和有目标没钱时该怎么办”的问题。

其次，慈善是他参与竞拍的另一大动力。段永平夫妇认为，钱财够用即可，因此将相当一部分财富捐赠给了慈善基金会。巴菲特午餐的竞拍款项全部用于慈善，这正与他的理念不谋而合。对于62.01万美元这个数字，他认为这顿饭是“无价的”，但凡事有度，这个价格恰好在他基金会的预算之内。面对外界关于为何捐款给国外而非国内的质疑，段永平解释道，美国对他而言已非“国外”，且他们在国内也一直默默捐款，只是不声张。他特别提到，他们曾在华东水灾时捐款，但后续缺乏反馈，这让他对国内的慈善环境有所保留。因此，他们选择在斯坦福大学设立基金会，专门资助中国学生，确保善款能有效利用，并与中国紧密相连。这个基金会不仅花钱，也由段永平亲自打理投资，回报丰厚，至今已有一倍多的成长。

2001年，段永平与妻子移居美国，他的投资生涯才真正拉开序幕。在此之前，他从未涉足股票市场。然而，短短五年间，他在美国股市赚到的钱，竟远超他在国内十多年企业经营的所得，个人财富的九成以上都来自股票投资。他谦虚地称自己“运气不错”，但无疑已是一位非常成功的投资者。

段永平对巴菲特投资真经的理解，核心在于“买股票就是买企业”。他认为，企业的价值在于其未来现金流的折现，如果当前股价低于其内在价值，便可买入。这个道理看似简单，但要准确估算企业价值却异常困难。他强调，简单不等于容易，大部分投资者往往受价格波动影响，而非深入了解企业。巴菲特的理论让他学会了在投资中坚持原则，不为外界噪音所动。

在众多投资中，网易无疑是段永平的“神来之笔”。2001年底，纳斯达克崩盘，中国网络概念股哀鸿遍野，搜狐、新浪、网易等股价跌至1美元附近，面临摘牌风险。段永平却在这片废墟中看到了黄金。他投入一百多万美元，甚至借贷，最终持有网易超过200万股，一度达到6%以上的股份。他坦言，这笔投资为他带来了近1亿人民币的回报。

他之所以选择网易，是因为他“看得懂”。作为“小霸王”游戏机的创始人，他对游戏行业有着深刻的理解。他认为网易是一家“本分”的公司，不追求快速扩张，没有“乱七八糟的事儿”。更重要的是，当时网易每股拥有两块多美元的现金，股价却不到一块钱，这在他看来是严重的低估。凭借做企业的经验，他坚信这家公司绝不可能一直亏损下去。他当时最大的遗憾，反而是网易股价没有继续下跌，这样他就能借钱买入更多。他深知，投资不能给自己设定目标，因为价格由市场决定，而价值是内在的。他所能做的，就是深入了解价值，然后耐心等待市场给予其公平的价格，通常这个等待期不会超过三年。他强调，只要找到好公司，有两三年的耐心，就能赚钱；反之，如果拿错了股票，时间越久亏损越大。

段永平的投资风格非常集中。他只投资自己能看懂的公司，就像巴菲特选择可口可乐而非微软一样。网易的成功，也印证了他的眼光。随着中国移动“移动梦网”的推出，短信业务为网易带来了丰厚利润；随后，网络游戏异军突起，成为网易最主要的盈利来源。网易股价一路飙升，一度逼近100美元大关，段永平的身价也因此暴增。

# Chapter 2: 从游戏到投资：段永平的商业智慧与人生哲学

段永平，这位传奇企业家，对投资有着自己独到的见解。他坦言，在众多投资标的中，网易是他最能“看懂”的企业之一。这并非偶然，因为他正是以游戏起家，曾一手打造出家喻户晓的“小霸王游戏机”，对游戏产业的脉络和潜力了如指掌。这份深刻的行业理解，让他对网易的商业模式和产品充满信心。

在投资策略上，段永平是“集中投资”的坚定拥护者。他回忆起当初投资网易的情景：最初投入一百多万美元，后来又借贷并利用融资，总计投入约两百万美元。事实证明，网易的股价表现并未让他失望。2001年，中国移动推出“移动梦网”后，短信业务迅速成为网易的盈利支柱；紧接着，网络游戏异军突起，成为其最主要的利润来源。网易公司因此获得了丰厚收益，其股价在二级市场上一飞冲天，一度逼近一百美元大关，段永平也因此次投资而身价暴增。

谈及选股原则，段永平强调最核心的一点是“了解企业”。他认为自己作为企业家出身，对企业的理解更为深刻，从不盲信所谓的“神话”或“故事”。他会根据观察到的情况，去想象企业的未来图景。当然，有时也会遇到“价值远超价格”的特殊情况，这便是巴菲特所说的绝佳买入时机。他以自己投资网易的经历为例，当初以八九毛钱甚至一块钱买入的股票，最终能持有到八九十块钱，这中间需要极大的定力。他深知，如果真正了解企业价值，就不会被每日波动的价格所困扰，否则，每天都会在卖与不卖之间挣扎。

段永平的投资范围也并非天生就熟悉，但他强调，在重仓投资或持有前，他一定会深入学习和了解。他投资的标的，大多与日常生活息息相关，对于那些“八杆子打不着、完全没有感觉”的领域，他绝不会轻易涉足，因为缺乏认知便无法做出判断。

面对亏损的股票，段永平的处理方式也体现了他的理性。他将投资视为投资一家企业：如果是好企业，当价格下跌时，若有余钱便可加仓；若无，则束之高阁，耐心等待。但若发现企业本质不佳，即使价格已跌去很多，但仍远高于其真实价值，那就应该果断卖出，因为不卖只会亏损更多。他总结道，最重要的是真正了解企业。

他为自己设定了严格的投资规则：“不该碰的，绝对不碰；该握在手里的，一定不放手。”他坚信，这些规则在任何经济行为中都适用，包括股票市场，这是他从巴菲特思想中领悟到的最大体会——“搞懂了，你就会赚钱。”目前，他在国内A股市场也有一只股票，即申万科，自2005年初买入并持有至今，已获得约100%的回报，这让他五年股龄的投资家身份更加实至名归。

回溯段永平的创业历程，同样充满了传奇色彩。1988年，他南下广东，接手了一家濒临破产的中山小厂。他毕业后曾在佛山工作，但对当时的工作不甚满意。当他得知中山有家公司在做游戏，而他本人又对游戏充满热情（甚至能玩到二十多级，让儿子都佩服不已），便觉得这是个有前途的领域。他亲自带领团队，挨家挨户地找客户，有时一天工作十几个小时，异常辛苦。正是这种亲力亲为的经历，让他意识到必须建立一套完善的系统，包括代理制、管理体系和价格体系，最终让小霸王在他离开时实现了逾十亿的年产值。

然而，作为“打工皇帝”的段永平并不满足于此。他提出改制，希望建立明确的员工奖励和股东回报机制，但怡华集团高层拒绝了他的要求。他认为，缺乏清晰的机制，企业难以长期发展。1995年7月，段永平毅然辞职。两个月后，他创立了步步高公司。起初，他并没有明确的创业方向，甚至曾想过远赴美国追求爱情。但许多因他而来、后又被小霸王新任管理者解雇的员工，再次找到他。出于责任感，段永平决定再次创业。他遵守承诺，一年内不做同行，转而从事外销业务，让大家有事可做。虽然第一年亏损不少，但他认为这是赢得了时间。这段经历为步步高日后在俄罗斯市场取得巨大成功（每年近两亿美元的销售额）奠定了基础。

从1997年开始，步步高公司在央视投放广告，并在两年后成为“央视标王”。段永平对此淡然处之，他认为央视广告在当时是最“便宜”的投入。对于外界“段永平只会做广告”的说法，他坚决反驳。他认为，一个企业如果只会做一样东西，注定会消亡。广告只是企业发展中的一块“短板”，需要补齐，但绝不能成为唯一的支柱。他举例说，曾有一款酒大肆投放广告，甚至成为“标王”，但因产品质量不佳，最终迅速消亡。这印证了他的观点：产品不好，广告打得再响也只是自寻死路。

如今，段永平在步步高所占股份很少，更多地扮演顾问角色。他定居美国，专注于投资和慈善事业，大部分时间陪伴家人，每年回中国几次，称之为“出差”。回顾过往，他坦言人生总会有遗憾，比如未能买到足够多的股票，或买错了股票导致亏损。但他认为，这些遗憾虽然存在，却不重要，因为人总会犯错，无法避免。他豁达地表示，生活中的波折和遗憾，正是构成人生精彩的一部分，也是值得铭记的记忆。
